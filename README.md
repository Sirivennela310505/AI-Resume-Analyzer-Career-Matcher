# AI Resume Analyzer & Career Matcher
This is a web-based student project that analyzes resumes and provides insights to help students understand their skills and career direction.
# Project Description
The AI Resume Analyzer takes a resume as input and:
- Identifies matched and missing skills
- Provides resume improvement recommendations
- Suggests suitable job roles
- Recommends possible career paths
This project is designed to help students make better career decisions.
# Features
- Resume analysis
- Skill matching and missing skills detection
- Smart resume insights and recommendations
- Career path suggestions
# Technologies Used
- HTML
- CSS
- JavaScript
# How to Use
1. Open the website
2. Upload or paste your resume
3. Click on analyze buttons
4. View skill analysis, recommendations, and career paths
